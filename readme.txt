Contributors: patrickgarman

Tags: reset password, mass password reset, password
Requires at least: 3.0
Tested up to: 3.04
Stable tag: 1.0

This plugin allows an administrator to reset all users passwords. Future versions will allow to choose what passwords to reset.


== Installation ==

Extract the zip file and just drop the contents in the wp-content/plugins/ directory of your WordPress installation and then activate the Plugin from Plugins page.
